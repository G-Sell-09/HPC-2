Performs a recursive multigrid method with pre- and post-smoothing using the v_cycle procedure.

Parameters
[in]Ngrid size
[in]ArrOfxmatrix with enough space to store x on all levels
[in]ArrOfbmatrix with enough space to store right hand sides on all levels
[in]ArrOfrmatrix with enough space to store residuals on all levels
[in]L# of levels for multigrid method
[in]L_tempcurrent level for rekursive call
[in]nu1number of iteration in the pre-smoothing operator
[in]nu2number of iteration in the post-smoothing operator
[in]wrelaxation factor for smoothing operator
[in]tolSmootolerance for the smoothing operator
[in]N_vecarray with values of N on each level
[in]N_potarray with values of N squared on each level
[in]lowermatrix L for the LU decomposition
[in]uppermatrix U for the LU decomposition
[in]smoothtypedef that points to the chosen smoothing operator
[in]mfMulttypedef that points to the chosen matrixfree mat-vec multiplication
[out]ArrOfxmatrix with enough space to store x on all levels
[out]ArrOfrmatrix with enough space to store residuals on all levels
